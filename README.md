# Blockchain-Based Botanical Traceability System

## Overview
This project is a blockchain-based supply chain system for Ayurvedic herbs.

It ensures:
- Traceability from harvest to product
- Tamper-proof records
- Public verification using blockchain

## Architecture
- Hyperledger Fabric → Core data storage
- Ethereum → Public hash verification
- Node.js → Backend API
- QR Code → Consumer verification
- SMS Gateway → Rural collector access

## Features
- Herb lifecycle tracking
- Role-based access (Collector, Processor)
- Blockchain replication (multi-peer)
- Hybrid blockchain (Fabric + Ethereum)
- QR-based verification
- SMS input system
